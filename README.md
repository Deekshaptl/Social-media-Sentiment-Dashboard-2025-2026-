1️⃣ Project Overview

The Social Media Sentiment Dashboard is a data analytics project that analyzes public opinions expressed on social media platforms. The dashboard helps understand how people feel about topics, brands, or products by classifying posts into positive, negative, or neutral sentiment.

Using data visualization, the dashboard allows users to track sentiment trends, analyze user engagement, and monitor public perception during the period 2025–2026.

This project helps businesses and organizations understand customer feedback and improve decision-making.

2️⃣ Objectives of the Project

The main objectives of this project are:

Analyze sentiment from social media posts

Identify positive, negative, and neutral opinions

Monitor sentiment trends over time

Understand user engagement patterns

Provide insights for business strategy and brand monitoring

4️⃣ Dataset Information

The dataset contains social media posts and related information.

Example columns in the dataset:

Column	Description
Post ID	Unique identifier for each post
Platform	Social media platform (Twitter, Instagram, Facebook)
Date	Date when the post was published
User Location	Geographic location of the user
Post Content	Text of the social media post
Sentiment	Positive, Negative, or Neutral
Likes	Number of likes received
Shares	Number of times the post was shared
Comments	Number of comments

This data helps analyze public opinion and engagement on social media.

5️⃣ Key Performance Indicators (KPIs)

The dashboard includes important metrics that summarize the data.

Examples:

Total Posts – Total number of analyzed social media posts

Positive Sentiment Percentage – Percentage of positive posts

Negative Sentiment Percentage – Percentage of negative posts

Neutral Sentiment Percentage – Percentage of neutral posts

Total Engagement – Total likes, shares, and comments

These KPIs help quickly understand the overall sentiment trend.

8️⃣ Key Insights

From the analysis, the following insights can be identified:

Positive sentiment may dominate discussions about successful campaigns.

Negative sentiment often increases during product issues or service complaints.

Engagement levels vary across social media platforms.

Some locations show stronger positive or negative opinions.

These insights help businesses understand customer perception and brand reputation.
